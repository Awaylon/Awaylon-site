# Awaylon 埃维隆空调出风口 - 网站规范文档

## 1. 项目概述

**项目名称**: Awaylon埃维隆艺术风口定制官方网站
**项目类型**: 品牌展示与产品销售网站
**核心功能**: 展示空调出风口产品系列，提供定制服务信息，支持在线询价
**目标用户**: 家装业主、设计师、工程商

---

## 2. UI/UX 规范

### 2.1 布局结构

**页面架构**:
- 固定顶部导航栏 (高度: 72px)
- 全屏视频主导英雄区 (100vh)
- 内容区域 (最大宽度: 1280px, 居中)
- 页脚 (联系信息、社交链接)

**响应式断点**:
- 移动端: < 768px (单列布局)
- 平板: 768px - 1024px (双列)
- 桌面: > 1024px (多列网格)

### 2.2 视觉设计

**主色调**:
- 主色: #0EA5E9 (气流蓝 - 品牌专用)
- 辅助色: #1F2937 (深炭灰)
- 强调色: #F59E0B (暖橙色，用于CTA按钮)
- 背景色: #FAFAFA (浅灰白)
- 深色背景: #111827 (近黑色)

**字体系统**:
- 中文标题: "Noto Sans SC", sans-serif (700, 900)
- 英文标题: "Montserrat", sans-serif (600, 700)
- 正文: "Noto Sans SC", sans-serif (400)
- 字号:
  - H1: 56px / 64px (桌面), 36px (移动)
  - H2: 40px / 48px
  - H3: 28px / 36px
  - 正文: 16px / 24px
  - 小字: 14px / 20px

**间距系统**:
- 基础单位: 8px
- 组件间距: 24px
- 区块间距: 64px (桌面), 48px (移动)
- 内容边距: 24px (桌面), 16px (移动)

**视觉效果**:
- 卡片阴影: 0 4px 24px rgba(0,0,0,0.08)
- 悬浮阴影: 0 8px 32px rgba(0,0,0,0.12)
- 边框圆角: 8px (按钮), 16px (卡片), 24px (大容器)
- 过渡动画: 0.3s cubic-bezier(0.4, 0, 0.2, 1)

### 2.3 组件规范

**导航栏**:
- 背景: 透明 (首页) / 白色 (滚动后)
- 字体: 14px, 字重 500
- 悬浮: 下划线动画 (#0EA5E9)

**按钮**:
- 主按钮: 背景 #0EA5E9, 白色文字, 高度 48px, 圆角 8px
- 次按钮: 边框 #1F2937, 透明背景, 高度 48px
- 悬浮: 亮度提升 + 轻微上移

**产品卡片**:
- 背景: 白色
- 边框: 1px solid #E5E7EB
- 悬浮: 边框变深 + 阴影扩展
- 图片比例: 4:3
- 标签: 角标显示"热卖"/"定制"

**表单**:
- 输入框: 高度 48px, 边框 #E5E7EB, 聚焦边框 #0EA5E9
- 文本域: 高度 120px
- 提交按钮: 全宽, 主色背景
